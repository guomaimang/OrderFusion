Course Information

    COMP3334: Computer Systems Security


      Topics to be covered

  • Overview: Security goals and policies, types of attacks, and threat models
  • Cryptography: Classical cryptography, public-key cryptography, and steganography
  • Authentication: Password systems, one-time passwords, strong password protocols, and password authentication protocols
  • Web Security: Input validation, SQL injection, cross-site scripting, cross-site request forgery, and non-validated redirects and forwards
  • Software Vulnerability: Buffer overflow, memory protection and corruption
  • Case study & Advanced topics: Blockchain, Merkle tree, zero-knowledge proof, and blind signatures
